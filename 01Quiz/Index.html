Quizz GitHub

1.¿Qué comando se utiliza para iniciar un repositorio?

  1. git status
  2. git add .
  3. git init
  4. git commit -m "first commit"

2.¿Qué comando se usa para cambiarle el nombre a la rama master a main?

  1. git branch - M main
  2. git add .
  3. git push - u origin main
  4. git status

3.¿Qué comando es para ver el estado del repositorio?

  1. git add .
  2. git status
  3. git remote - v
  4. git init

4.¿Qué comando se usa para agregar cambios?

  1. git add .
  2. git commit - m "first commit"
  3. git remote - v
  4. git status

5.¿Qué comando se usa para subir los cambios a GitHub?

  1. git commit - m "first commit"
  2. git push - u origin main
  3. git add .
  4. git remote - v

6.¿Comando para entrelazar repositorio local con la nube?

  1. git remote add ...
  2. git push - u origin main
  3. git add .
  4. git remote - v

7.¿Qué comando se usa utiliza para entrar a una carpeta?

  1. git commit - m "first commit"
  2. cd
  3. add
  4. git init

8.¿Qué comando es para clonar un repositorio?

  1. git commit - m "first commit"
  2. git push - u origin main
  3. git clonar
  4. git clone

9.¿Qué comando se usa para configurar el nombre?

  1. git commit - m "first commit"
  2. git config --global user.name ""
  3. git config --global user.email
  4. git remote - v

10.¿Comando para cambiar la URL del repositorio?

  1. git set url- add
  2. git set -url add
  3. git URL add
  4. git set add
